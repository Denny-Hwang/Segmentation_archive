## How to Use

- Start with the **reading roadmap** if you are new to segmentation
- Use the **glossary** as a quick reference for terminology
- Check **conference deadlines** for submission planning
- Import **bibliography.bib** into your LaTeX or reference manager
- Browse **awesome links** for tutorials and community resources
