# Key Equations in U-Net

## Weighted Cross-Entropy Loss

U-Net uses a pixel-wise weighted cross-entropy loss to handle class imbalance and emphasize cell boundaries:

`L = Σ_x w(x) · log(p_class(x)(x))`

where `p_class(x)(x)` is the softmax probability of the correct class at pixel x, and `w(x)` is a pre-computed weight map.

## Weight Map Computation

The weight map balances class frequencies and emphasizes boundaries between touching cells:

`w(x) = w_c(x) + w_0 · exp(-(d₁(x) + d₂(x))² / (2σ²))`

where:
- `w_c(x)` is the class frequency balancing weight (inverse of class frequency)
- `d₁(x)` is the distance to the border of the nearest cell
- `d₂(x)` is the distance to the border of the second nearest cell
- `w₀ = 10` and `σ ≈ 5` pixels are hyperparameters

The exponential term assigns high weights to pixels between closely touching cells, forcing the network to learn thin separating boundaries. This is critical for instance segmentation in biomedical images where cells frequently touch.

## Softmax Function

Per-pixel classification uses the softmax function:

`p_k(x) = exp(a_k(x)) / Σ_{k'} exp(a_{k'}(x))`

where `a_k(x)` is the activation of feature channel k at pixel position x.

## Dice Coefficient

While not used in the original U-Net, the Dice coefficient (introduced by V-Net) became the standard metric for medical segmentation:

`DSC = 2|X ∩ Y| / (|X| + |Y|)`

where X is the predicted segmentation and Y is the ground truth. As a loss function:

`L_Dice = 1 - (2·Σ p_i·g_i + ε) / (Σ p_i + Σ g_i + ε)`

where p_i are predicted probabilities, g_i are ground truth labels, and ε is a smoothing factor to prevent division by zero.

## Data Augmentation Transforms

The paper emphasizes elastic deformation as the most important augmentation:

`(x', y') = (x + Δx(x,y), y + Δy(x,y))`

where Δx and Δy are smooth random displacement fields generated by:
1. Sampling random values from a uniform distribution on a coarse grid
2. Smoothing with a Gaussian filter (σ = 10 pixels)
3. Scaling by a deformation strength parameter (α = elastic deformation magnitude)

Additional augmentations: random rotations, random horizontal/vertical flips, random scaling, and random brightness/contrast adjustments. The paper reports that elastic deformation is essential for achieving good performance with very few training images (30 annotated images for the ISBI challenge).

## Overlap-Tile Strategy

For seamless segmentation of large images, U-Net uses an overlap-tile strategy:

1. The input image is divided into overlapping tiles
2. Each tile is padded with mirrored border pixels
3. Predictions are made for each tile
4. Only the central (non-overlapping) region of each prediction is kept
5. Predictions are stitched together to form the full segmentation map

This allows segmentation of arbitrarily large images with constant memory usage.
