# seg-research-archive

**U-Net Series & Image Segmentation Research Archive**

Paper Reviews | Code Analysis | Reverse Engineering | Reproduction Experiments | Interactive Explorer

---

## About

A systematic research archive for the image/video segmentation domain.
Starting from U-Net (2015) to current SOTA models (SAM2, Mask2Former, etc.),
this repository explores the evolution of segmentation architectures through
paper reviews, code reverse engineering, reproduction experiments, and visualizations.
